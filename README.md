# GBA Emulator

A Game Boy Advance emulator written in Rust.

## About

This project is a GBA emulator built as an exercise in low-level systems programming in Rust. The goal is to gain hands-on experience with topics like:

- CPU emulation (ARM7TDMI)
- Memory mapping and bus architecture
- Hardware register I/O
- Bit manipulation and binary arithmetic

## Status

Work in progress.

## Building

```bash
cargo build
```

## Running

```bash
cargo run
```
